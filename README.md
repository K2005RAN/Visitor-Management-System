# Industrial Security Visitor Pass System

A robust, full-stack **MERN platform** designed to orchestrate, audit, and track the security lifecycles of visitors across industrial plant locations. 

This system bridges the gap between traditional manual gate registers and automated corporate security logistics, providing real-time data verification, instantaneous access pass generation, and pre-mapped identity tracking.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS, Vite runtime engine, Lucide Icons.
* **Backend:** Node.js, Express.js (RESTful API architecture layer).
* **Database:** MongoDB mapped via Mongoose Object Data Modeling (ODM).

---

## 📦 Project Structure

This repository contains the core application source code:
* **`Backend/`**: The Node.js and Express.js API application service layer, database schemas, and routing logic.
* **`Frontend/`**: The React.js  application for  client user interface.
